# Retail Demand & Inventory Analytics

## Overview

This project develops a retail analytics framework using transactional POS data to simulate decision-support insights for demand forecasting, inventory optimization, promotion analysis, and product association mining.

## Dataset

Online Retail dataset containing ~540K transactions across multiple SKUs and countries.

## Objectives

* Analyze SKU-level demand variability
* Evaluate inventory turnover behavior
* Construct strategic demand–inventory segmentation
* Detect promotion-driven demand spikes
* Discover product co-purchase patterns

## Methodology

* Coefficient of variation used for demand stability classification
* Turnover proxy derived from quantity-to-transaction ratio
* Baseline daily demand modeled for uplift detection
* Invoice-level self-join applied for market basket mining

## Key Insights

* Significant proportion of SKUs exhibit erratic demand patterns
* Mixed inventory velocity suggests differentiated replenishment strategies
* Promotion spikes reveal episodic demand surges
* Strong product pair associations highlight bundling opportunities
